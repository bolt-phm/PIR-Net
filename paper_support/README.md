# Paper Support Assets

This directory contains reproducibility-oriented assets used for manuscript figure generation and presentation support.

## Contents

1. `py_round2_figures/`: Python scripts for publication figure/table generation (historical folder name retained for compatibility).
2. `figures/`: generated figure assets used by the manuscript workflow.
3. `py/`: supplementary visualization utilities.

## Notes

1. These assets are provided for research transparency and manuscript reproduction.
2. Core model training/evaluation pipelines remain under `experiments/`.
