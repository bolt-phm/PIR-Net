# Code of Conduct

## 1. Commitment

This project is committed to maintaining a respectful, inclusive, and technically constructive collaboration environment.

## 2. Expected Behavior

1. Communicate professionally and respectfully.
2. Ground technical claims in reproducible evidence.
3. Provide actionable feedback focused on code, data, and methods.
4. Resolve disagreements through transparent technical discussion.

## 3. Unacceptable Behavior

1. Harassment, personal attacks, or discriminatory language.
2. Repeated disruptive behavior that blocks constructive collaboration.
3. Sharing private credentials, sensitive data, or personal information.
