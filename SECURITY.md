# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| main (latest) | Yes |
| historical commits | No |

## Reporting a Vulnerability

Please report security issues privately by email:

- Contact: 2483839342@qq.com
- Subject template: `[PIR-Net Security] <short title>`

Please include:

1. A clear vulnerability description.
2. Reproduction steps or proof of concept.
3. Potential impact assessment.
4. Suggested mitigation (if available).

We will acknowledge receipt as soon as possible and coordinate a fix before public disclosure.
