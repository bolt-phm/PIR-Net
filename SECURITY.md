# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| `main` (latest) | Yes |
| Historical commits | No |

## Reporting a Vulnerability

Please report security issues privately by email:

- Contact: `2483839342@qq.com`
- Subject: `[PIR-Net Security] <short title>`

Please include:

1. A concise vulnerability description.
2. Reproduction steps or proof of concept.
3. Impact assessment.
4. Suggested mitigation, if available.

Reports will be acknowledged and handled through coordinated disclosure.
